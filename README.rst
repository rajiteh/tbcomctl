===================================
Common Controls Library for Telebot
===================================

Library provides common controls to use with Telebot_ library.

Controls:

* Picklist
* Rating
* Post Buttons
* Keyboard

Abstractions:

* Form (combines)

Utilities:

* Subscription check
* Middleware
* Helper functions for logging.
